# dbt + Snowflake E-Commerce Analytics Platform

**Status:** In progress — Day 1 / 14

Production-grade analytics platform on the Brazilian E-Commerce (Olist) dataset, demonstrating modern data stack patterns used at tier-2 tech companies (Stripe, Capital One, Atlassian).

## Stack
- **Warehouse:** Snowflake (Standard, AWS us-west-2)
- **Transformation:** dbt Core
- **Visualization:** Tableau Public
- **CI/CD:** GitHub Actions
- **Modeling:** Kimball star schema, medallion architecture (raw -> staging -> marts)

Project 1 in a 13-project portfolio. More details coming as the project develops.
